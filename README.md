# Test Case Analysis

This repository contains the initial TypeScript scaffold for the `local.test-case-analysis` VS Code extension.

## Current state

- Adds a `Test Case Analysis` view container to the Activity Bar
- Shows a sidebar view with a `Run Hello World` item
- Registers the `Test Case Analysis: Run Hello World` command
- Serves as the starting point for the real project functionality

## Development

Prerequisite: install Node.js so `npm` is available.

1. Run `npm install` once
2. Press `F5` in VS Code
3. Click the `Test Case Analysis` icon in the Activity Bar
4. Select `Run Hello World` in the sidebar
